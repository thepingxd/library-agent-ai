# TAGS — Vocabulário controlado

Estas são as **únicas** tags válidas na biblioteca. Não invente tags novas por
arquivo: se faltar algo, adicione aqui primeiro. Isso mantém a busca consistente.

## Tipo (uma por item — reflete a pasta onde ele vive)

| Tag           | Pasta                  | O que é                                        |
|---------------|------------------------|------------------------------------------------|
| `template`    | `templates/`           | Modelo de site/página/função reaproveitável    |
| `boilerplate` | `boilerplates/`        | Scaffold completo de projeto                    |
| `component`   | `components/`          | UI isolada (React, Vue, etc.)                   |
| `snippet`     | `snippets/`            | Trecho curto de código                          |
| `logo`        | `brand/logos/`         | Logo/marca                                      |
| `colors`      | `brand/colors/`        | Paleta / tokens de cor                          |
| `font`        | `brand/fonts/`         | Fonte tipográfica                               |
| `image`       | `assets/images/`       | Foto, textura, background raster                |
| `illustration`| `assets/illustrations/`| Ilustração / vetor                              |
| `icon`        | `assets/icons/`        | Ícone                                           |
| `doc`         | `references/`          | Documentação, cheatsheet, exemplo               |
| `prompt`      | `prompts/`             | Prompt reaproveitável                           |
